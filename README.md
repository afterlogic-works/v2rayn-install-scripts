# 🚀 v2rayn-install-scripts

**Cross-platform scripts for automated installation and configuration of [v2rayN](https://github.com/2dust/v2rayN)**

> _⚠️ This project is not affiliated with the official v2rayN._

---

## ✨ Features

- 🛠️ Fully automated installation  
- ⚙️ Auto-configuration with your custom parameters  
- 🖥️ Desktop shortcut creation (where supported)  
- 🧘 Zero manual steps — just run the script and relax  

---

## ⚡ Quick Start

Just copy and paste the appropriate command into your terminal — the script will take care of the rest! 🎉

### 🪟 Windows

Open **PowerShell as Administrator** and run:

```powershell
irm https://raw.githubusercontent.com/afterlogic-works/v2rayn-install-scripts/main/scripts/windows/windows.ps1 | iex
```

---

### 🍏 macOS

#### ✅ Apple Silicon (M1, M2, M3 and newer)

```bash
bash <(curl -sSL https://raw.githubusercontent.com/afterlogic-works/v2rayn-install-scripts/main/scripts/macos/macos-arm.sh)
```

#### 💻 Intel-based Macs

```bash
bash <(curl -sSL https://raw.githubusercontent.com/afterlogic-works/v2rayn-install-scripts/main/scripts/macos/macos-intel.sh)
```

---

### 🐧 Linux (Debian-based systems only)

```bash
wget -qO- https://raw.githubusercontent.com/afterlogic-works/v2rayn-install-scripts/main/scripts/linux/linux-x64.sh | bash
```

> 💡 You may need to use `sudo` if installation requires administrative privileges.
